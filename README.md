# data-analysing-assigment
Analyze Employee &amp; Student datasets using Python. Features comprehensive data understanding, custom feature engineering, and deep analytical insights. Includes professional visualizations using Matplotlib and Seaborn to showcase advanced data manipulation and analytical storytelling in a structured Jupyter Notebook. 
In this assignment, students will work with two datasets:

Employee Dataset
Student Dataset
The objective is to perform data understanding, create new meaningful features (columns), extract analytical insights, and visualize the results using:

Matplotlib
Seaborn
Students are expected to demonstrate strong analytical thinking, proper data manipulation skills, and professional-quality visualizations.
